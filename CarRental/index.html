Car Rental

Beschreibung

Bei CarRental handelt es sich um eine REACT Anwendung, die im Rahmen der Veranstaltung Programmierung REACT an der Hochschule Furtwangen entwicklet wurde. In der Anwendung können Fahrzeuge beispielhaft gemietet und vermietet werden.

Technologien

  • React

Tätigkeiten

  • Erstellung des Designs
  • Aufbau der Datenbank
  • Umsetzung des Client seitigen Codes

Funktionen

  • Anlegen von Autos
  • mieten von Autos
  • vermieten von Autos
